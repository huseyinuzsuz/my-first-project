CORDOVA


Diller : HTML, CSS, JavaScript
Hedef isletim Sistemi : Cross-platform, Android, iOS
Kullanilabildigi isletim Sistemleri : Windows , MacOS , Linux, Android, Windows Phone
Kullanici Kitlesi : Web gelistiricileri
Lisans turu : Apache 2.0 Lisansi
Fiyat : 350€
Eski adi PhoneGap olan Apache Cordova , web gelistiricilerinin HTML, CSS ve JavaScript kullanarak mobil uygulama geliştirmesine olanak sağlayan açık kaynaklı Cross-Platform mobil uygulama geliştirme çerçevesidir. Nitobi Software (şuan ki Adobe) firması tarafından geliştirilmiştir.
Artıları : HTML5, CSS3 ve JavaScript ile geliştirme yapabilirsiniz. Cross-Platform desteği sağlar. Öğrenim kolayligi sunar. Hizli test edebilmeyi sağlar. Tamami ucretsiz ve acik kaynalidir. Apache 2.0 lisansi altinda yayinlanmıştir, App Store, Play Store ve daha fazlasında uygulama içi satın alımları destekler. Platformlar arası mobil uygulama yazılabilir. Eksileri :
Grafik yoğunluklu uygulamalarda performans sorunları oluşabilir. Kullanıcı arayüzü pencere öğelerinin eksikliği sorun oluşturabilir. Cordova hakkında daha fazla bilgi için buraya tıklayabilirsiniz.
